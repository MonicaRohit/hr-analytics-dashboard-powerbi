# HR Data Analysis Dashboard | Workforce, Attrition & Hiring Insights

🔷 Overview
This project presents an interactive HR Analytics dashboard built in Power BI to analyze workforce distribution, attrition patterns, and hiring trends.

The dashboard helps answer key business questions:
- Where is the workforce concentrated?
- Which groups are at higher attrition risk?
- What hiring sources are most effective?
- How has workforce grown over time?

📊 Key Metrics
- Total Headcount: 311
- Active Employees: 207
- Attrition Count: 104
- Attrition Rate: 33.44%
- Average Salary: 69.02K
- Average Age: 47.41

📈 Dashboard Features

🔹 Workforce Distribution
- Headcount by Department shows heavy concentration in Production (~67%)
- Age group analysis highlights majority workforce in 36–45 range
- Marital status + gender breakdown for demographic insights

🔹 Attrition Analysis
- Attrition trends by hiring year identify peak exits (2011–2012)
- Attrition rate by recruitment source reveals high-risk hiring channels
- Age-based attrition analysis identifies older age groups (55+) with highest attrition %

🔹 Hiring Insights
- Most employees hired through job portals (Indeed, LinkedIn)
- Lower contribution from niche or alternative hiring sources
- Helps evaluate hiring strategy effectiveness

🔹 Workforce Growth
- Cumulative headcount trend shows steady organizational growth
- Hiring spikes align with higher attrition cohorts

🎯 Key Business Insights

- Workforce is highly concentrated in Production (~67%), indicating operational dependency
- Attrition rate is highest among employees aged 55+, followed by 46–55
- Employees hired via Google Search show highest attrition rate despite lower hiring volume
- Peak attrition is linked to employees hired during 2011–2012
- Core workforce lies in the 36–45 age group, forming organizational backbone

🛠 Tools & Technologies

- Power BI (Dashboard & Visualization)
- DAX (Calculated Measures)
- Excel / CSV (Data Source)
